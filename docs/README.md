# Documentation

Public documentation for this repository:

- `DATASETS.md`: dataset sources, download notes, and required local folder
  structures.
- `ASSETS.md`: checkpoint/result asset policy and what is committed to Git.
- `benchmark_curation_requirements.md`: maintainer checklist for adding or
  updating benchmark methods.

Internal audit reports, local migration notes, and machine-specific link
inventories are intentionally excluded from Git by `.gitignore`.
